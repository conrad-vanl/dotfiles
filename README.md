Drew Barontini's Dotfiles
=========================

Installation
------------

```bash
curl -O https://raw.githubusercontent.com/drewbarontini/dotfiles/master/install.sh
bash install.sh
```

Once the script runs, you can `rm install.sh`.

What's Included
---------------

- Git
- Bash
- Sublime Text
- Vim
